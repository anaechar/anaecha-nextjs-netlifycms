1. Go to [Metamask.io](https://metamask.io)

![Metamask Wallet Website](https://res.cloudinary.com/beforeifly/image/upload/v1627876319/anaecha.com/posts/metamask-website2_oagepn.jpg "Metamask Wallet Website")

2. Click on **Download** button

3. Select **Chrome** from the tab and Click on **Install Metamask for Chrome**

![Metamask Chrome Extension](https://res.cloudinary.com/beforeifly/image/upload/v1627876833/anaecha.com/posts/metamask-chrome-extension_zatfpb.jpg "Metamask Chrome Extension")
